# Exercício 2: Interface para Formas Geométricas

## Descrição:

Implemente um sistema que calcule área e perímetro de diferentes formas
geométricas.

## Requisitos:

### 1. Criar a interface FormaGeometrica com os métodos:

o calcularArea().
o calcularPerimetro().

### 2. Criar duas classes que implementam FormaGeometrica:

o Retangulo (atributos: largura e altura).
o Circulo (atributo: raio).

### 3. Criar um programa que instancie um Retangulo e um Circulo, solicite os valores ao usuário e exiba os cálculos da área e perímetro.
